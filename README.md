# Supreethraj_Task16

## Project Overview
This project is a small web app that shows student cards on a page. It uses `index.html` for the page, `style.css` for the design, and `script.js` to show the student cards and let you search by name.

## Learning Objectives
* Learn how to make cards for students using HTML.
* Use CSS to style the cards and page layout.
* Use JavaScript to create cards dynamically from a list of student objects.
* Learn how to filter the cards by typing in the search box.
* Practice using `document.getElementById`, loops, and functions.

## Relevant Files
* `index.html` - The main web page structure.
* `style.css` - The styling for the cards and page.
* `script.js` - The code that creates student cards and search behavior.

## How to Run
1. Open the `Supreethraj_task(16)` folder on your computer.
2. Make sure `index.html`, `style.css`, and `script.js` are all in the same folder.
3. Double-click `index.html` to open it in your browser.
4. Type a student name in the search box to filter the cards.

## Reflection & Key Takeaways
I learned how JavaScript can make HTML content appear on the page by creating elements in `script.js`. The search feature uses a filter to show only students whose names match what I type.
